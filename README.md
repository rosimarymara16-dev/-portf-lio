# Portfólio Profissional - Rosimary

Portfólio pessoal desenvolvido com HTML5, CSS3 e JavaScript, apresentando projetos e habilidades de forma responsiva e moderna.

## 📋 Descrição do Projeto

Portfólio profissional minimalista com design em preto e branco, destacando:

- **Estrutura Semântica HTML5**: Cabeçalho com menu de navegação, seções "Sobre Mim" e "Meus Projetos", e rodapé com informações de contato
- **Estilização Moderna**: Utilização de Flexbox e CSS Grid para layouts responsivos
- **Design Responsivo**: Totalmente adaptado para dispositivos móveis, tablets e desktops
- **Interatividade JavaScript**: Array de objetos com projetos renderizados dinamicamente no DOM
- **Experiência do Usuário**: Scroll suave, menu mobile interativo e animações em hover

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6+)
- Git & GitHub
- GitHub Pages

## ✨ Funcionalidades

- Hero section impactante com tipografia de destaque
- Menu de navegação responsivo com efeitos hover modernos
- Renderização dinâmica de projetos via JavaScript
- Cards de habilidades com animações interativas
- Layout totalmente responsivo
- Design minimalista em preto e branco
- Fontes Google (Inter + Playfair Display)
- Animações suaves e transições elegantes

## 📝 Personalização

Antes de fazer o deploy, personalize o conteúdo:

### No arquivo `index.html`:
- Altere as informações da seção "Sobre Mim"
- Atualize os dados de contato no rodapé
- Modifique os links das redes sociais

### No arquivo `script.js`:
- Edite o array `projetos` com seus próprios projetos
- Adicione ou remova projetos conforme necessário
- Atualize os links dos projetos

### No arquivo `styles.css`:
- Personalize as cores e gradientes
- Ajuste espaçamentos e tamanhos conforme preferir


## 📱 Responsividade

O site é responsivo e se adapta a:
- **Desktop**: Layout completo com grid de 3 colunas
- **Tablet** (768px): Layout ajustado com 2 colunas
- **Mobile** (480px): Layout de coluna única com menu mobile
